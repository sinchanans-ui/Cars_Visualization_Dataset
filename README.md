# 🚗 Cars Dataset Visualization

## 📊 Overview
This project focuses on analyzing and visualizing car dataset information using Python and Power BI.  
It explores relationships between car features such as price, mileage, horsepower, and fuel efficiency to uncover key insights.

## 🧠 Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Power BI for interactive dashboards
- Excel for data preprocessing

## 📁 Dataset
The dataset contains details of various car models, including:
- Manufacturer and model
- Engine specifications
- Mileage and fuel type
- Price and performance metrics

## 📈 Key Insights
- Correlation between engine size and car price  
- Mileage comparison across fuel types  
- Top manufacturers by average price and performance  

## 🔄 Project Workflow
1. Data cleaning and preprocessing using Python.  
2. Exploratory data analysis and visualization with Matplotlib and Seaborn.  
3. Dashboard creation in Power BI for interactive insights.
